# Chakchak Works Homebrew Tap

Small Mac tools that snap right in — by [ulBible](https://github.com/ulBible).

```bash
brew install --cask ulBible/tap/vclips     # clipboard manager
brew install --cask ulBible/tap/badasseo   # Korean-first local voice typing
```

Apps auto-update via Sparkle after install; `brew upgrade --cask` works too.
